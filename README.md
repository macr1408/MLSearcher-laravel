# Mercadolibre Deals Finder

This is a tool for searching items within Mercadolibre, but allowing us to combine filters and save our preferences.

Live demo: https://mldealsfinder.herokuapp.com/

### Technologies used

- Laravel
- Vue Js
- Tailwind
- Axios
- Placeholder loading
